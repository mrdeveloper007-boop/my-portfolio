# Abdul Qadeer — Portfolio

Personal portfolio of **Abdul Qadeer**, a Shopify Full Stack Developer based in Lahore, Pakistan.

**Live:** [my-portfolio-shopify.vercel.app](https://my-portfolio-shopify.vercel.app/)

## Stack

React 16 · SCSS · CRA · Manrope + Space Grotesk · FontAwesome.

Single-page architecture. All content in `src/data.js`, all sections in `src/App.js`, all styles in `src/App.scss`.

## Run locally

```bash
npm install
npm start
```

Opens on `http://localhost:3000`.

## Build

```bash
npm run build
```

Outputs to `build/` — deploy that folder to any static host (Vercel, Netlify, GitHub Pages, S3).
